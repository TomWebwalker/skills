name: qa-local
description: Guid developers through the process of running tests locally in the browser
---

1. Ask if the code is done and ready for testing
2. If not, end the conversation and ask to come back when it's ready
3. If yes, check if the ticket has a comment for QA instructions
4. If not, write a comment with instructions for QA testing.
5. If there are instructions, guide me through the steps to run the tests locally in the browser, including any necessary setup or commands to execute.
6. After each step, ask if I have completed and if there are any issues fix them with the angular keeper.
